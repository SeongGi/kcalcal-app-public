# 안드로이드 USB 연결 테스트 가이드 (Chrome Port Forwarding)

Vercel 배포 없이, PC에 스마트폰을 USB로 연결하여 바로 테스트하는 방법입니다.
이 방법을 사용하면 **HTTPS 없이도 카메라 기능이 작동**합니다 (폰에서 `localhost`로 인식하기 때문).

## 준비물
- 안드로이드 스마트폰
- USB 케이블
- PC에 설치된 Chrome 브라우저

## 1단계: 스마트폰 설정 (USB 디버깅 켜기)
1. 폰의 **설정 > 휴대전화 정보 > 소프트웨어 정보**로 이동
2. **빌드 번호**를 7번 연속 터치하여 "개발자 모드" 활성화
3. 뒤로 가서 **설정 > 개발자 옵션** 메뉴 진입
4. **USB 디버깅** 스위치 켜기

## 2단계: PC와 연결 및 설정
1. 폰을 PC에 USB로 연결 (폰 화면에 "USB 디버깅 허용" 팝업이 뜨면 **허용** 선택)
2. PC에서 Chrome 브라우저 주소창에 `chrome://inspect/#devices` 입력 후 엔터
3. **Devices** 섹션에 연결된 폰 모델명이 보이는지 확인
4. `Port forwarding` 버튼 클릭
5. 설정창에 다음 내용 추가 (이미 있으면 확인만):
   - **Port**: `3000`
   - **IP address and port**: `localhost:3000`
6. `Enable port forwarding` 체크박스 체크 → **Done** 클릭

## 3단계: 앱 실행 및 테스트
1. PC 터미널에서 로컬 서버가 실행 중이어야 합니다 (`npm run dev`)
2. 폰에서 **Chrome 브라우저** 열기
3. 주소창에 `localhost:3000` 입력
4. PC에서 실행 중인 앱이 폰 화면에 뜹니다!
5. **스캔 시작하기**를 눌러 카메라 권한을 허용하고 기능을 테스트하세요.

> **참고**: 이 상태에서 폰의 Chrome 메뉴(⋮)를 눌러 **"앱 설치"** 또는 **"홈 화면에 추가"**를 하면 아이콘이 생성되어 진짜 앱처럼 쓸 수 있습니다.
