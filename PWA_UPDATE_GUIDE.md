# PWA 업데이트 가이드

## 📱 핸드폰에서 앱 업데이트 방법

### 자동 업데이트 (권장)

PWA는 백그라운드에서 자동으로 업데이트를 확인합니다:

1. **앱 재실행**
   - 앱을 완전히 종료
   - 다시 실행
   - Service Worker가 자동으로 새 버전 다운로드

2. **확인 방법**
   - 설정 페이지에서 버전 확인
   - 새로운 기능이 보이면 업데이트 완료

### 수동 업데이트

자동 업데이트가 안 될 경우:

#### iOS (Safari)
1. 앱 열기
2. 아래로 당겨서 새로고침 (Pull to Refresh)
3. 또는 Safari에서 URL 직접 접속 후 재설치

#### Android (Chrome)
1. 앱 열기
2. 메뉴(⋮) → 새로고침
3. 또는 Chrome에서 URL 직접 접속

### 강제 업데이트

업데이트가 안 될 경우:

1. **캐시 삭제**
   - 브라우저 설정 → 사이트 데이터 삭제
   - 앱 재설치

2. **앱 재설치**
   - 홈 화면에서 앱 아이콘 삭제
   - 브라우저에서 사이트 접속
   - "홈 화면에 추가" 다시 실행

## 🔄 Service Worker 업데이트 주기

- **자동 확인**: 앱 실행 시마다
- **백그라운드 업데이트**: 24시간마다
- **즉시 적용**: 앱 재시작 시

## ⚠️ 주의사항

- 업데이트 중에는 인터넷 연결 필요
- 데이터는 IndexedDB에 저장되어 유지됨
- 백업 기능으로 데이터 안전하게 보관 가능
